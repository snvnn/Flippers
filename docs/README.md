## 아카이브 사용 규칙

- 현재 참고 문서만으로 해결되지 않는 부분이 있을 때만 아카이브 문서를 확인한다.
- 과거 판단 근거, 세부 구현 로그, 이전 리뷰 문구, 긴 초안 원문이 필요할 때만 아카이브를 연다.
- 현재 작업 판단은 항상 `Current` 문서와 실제 코드 상태를 우선한다.
