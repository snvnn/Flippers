## 1. 총칙

`Flippers`는 일본어 단어 및 한자 학습을 위한 iOS/iPadOS 애플리케이션입니다. 본 개인정보처리방침은 `Flippers`가 어떤 정보를 수집하고, 어떤 목적으로 사용하며, 어떤 외부 서비스와 연동하는지 설명합니다.

운영자 정보, 사업자 정보, 연락처, 개인정보 보호책임자 정보는 출시 전 확정 후 본 문서에 반영되어야 합니다.

## 2. 수집하는 정보

### 2.1 사용자가 직접 제공하는 정보

- 이메일 주소
- 비밀번호 기반 로그인 정보
- Apple 로그인을 통해 제공되는 계정 정보
- 사용자가 직접 입력한 카드, 덱, 섹션, 예문, 메모 등 학습 데이터

### 2.2 서비스 이용 과정에서 생성되는 정보

- 카드 생성 시각, 복습 시각
- SRS 학습 상태 값
- 학습 평가 기록

### 2.3 OCR 기능 사용 시 처리되는 정보

- 사용자가 선택하거나 촬영한 이미지
- 기기 내 OCR로 추출된 텍스트
- 사용자가 `Claude 읽기/뜻 보완` 옵션을 켠 경우, 읽기와 뜻 보완을 위한 이미지 또는 텍스트

## 3. 개인정보의 이용 목적

`Flippers`는 아래 목적 범위에서만 개인정보를 처리합니다.

- 회원가입, 로그인, 계정 식별
- 카드 생성 및 학습 기능 제공
- SRS 상태 저장 및 복습 기록 관리
- 기기 간 데이터 동기화
- OCR 기반 카드 생성
- 사용자가 선택한 경우 외부 AI를 이용한 읽기 및 뜻 보완
- 오류 확인 및 서비스 안정성 개선

## 4. 외부 서비스 및 제3자 처리

`Flippers`는 다음 외부 서비스를 사용할 수 있습니다.

### 4.1 Apple

- Sign in with Apple
- CloudKit
- Photos / Camera 권한

사용 목적:

- Apple 로그인 제공
- 학습 데이터 동기화
- 사용자가 선택한 이미지 접근 및 촬영

### 4.2 Google Firebase

- Firebase Authentication

사용 목적:

- 이메일/비밀번호 로그인
- Apple 로그인 연동 계정 처리

### 4.3 Anthropic Claude API

사용 목적:

- OCR 결과의 읽기 및 뜻 보완

중요:

- 이 기능은 기본적으로 꺼져 있습니다.
- 사용자가 앱 내에서 해당 옵션을 켠 경우에만 기기 내 OCR로 추출된 텍스트가 외부 AI 서비스로 전송될 수 있습니다.
- 원본 OCR 이미지는 앱에서 외부 AI 서비스로 직접 전송하지 않습니다.
- 민감정보, 타인의 개인정보, 금융정보 등이 포함된 이미지는 업로드하지 않는 것을 권장합니다.

## 5. 국외 이전 가능성

외부 서비스 제공자의 인프라 위치에 따라 개인정보가 대한민국 외 지역에서 처리될 수 있습니다.

- Apple CloudKit
- Google Firebase Authentication
- Anthropic Claude API

국외 이전 관련 세부 항목은 실제 운영 계정, 리전 설정, 계약 상태를 반영하여 출시 전에 확정해야 합니다.

## 6. 보유 및 이용 기간

원칙적으로 개인정보는 수집 및 이용 목적이 달성될 때까지 보관합니다.

현재 초안 기준 보관 원칙은 다음과 같습니다.

- 계정 정보: 계정 유지 기간 동안 보관
- 카드, 덱, 학습 기록: 계정 유지 기간 동안 보관
- OCR 이미지: 장기 보관을 기본값으로 하지 않으며, 기능 처리에 필요한 범위에서만 사용
- OCR 보완용 외부 전송 데이터: 사용자가 해당 기능을 켠 경우에만 처리

법령상 보관 의무가 있는 경우에는 해당 법령에 따른 기간 동안 별도로 보관할 수 있습니다.

## 7. 이용자의 권리

이용자는 관련 법령이 정하는 범위에서 다음 권리를 행사할 수 있습니다.

- 개인정보 열람 요청
- 정정 요청
- 삭제 요청
- 처리 정지 요청

현재 앱은 일부 권리 기능이 앱 내에서 완전 자동화되어 있지 않을 수 있으며, 출시 전 권리 행사 채널과 절차를 확정해야 합니다.
